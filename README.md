## Overview

This repository contains the complete source code, Conda environment specification, and generated figures for the dissertation:

**"Methodologies for Sorption‑Based Atmospheric Water Harvesting to Cater to Freshwater Use in Marine Survival Crafts"**

**Author:** Anirban Das  
**Programme:** Extra Master (Part D – Dissertation)  
**Institution:** Maritime Training Institute  
**Date:** April 2026

The code implements the **Dubinin–Astakhov (D‑A) thermodynamic isotherm** to model equilibrium water uptake and daily water yield for multiple sorbent materials under representative maritime conditions (relative humidity 60–90%, temperature 15–35°C). The study evaluates four sorbents:

- Silica Gel (baseline)
- LiCl‑Composite (salt‑based)
- Biomass Aerogel (high‑yield, sustainable)
- MOF‑303 (metal‑organic framework)

All calculations are desk‑based, use only open‑access data (NASA POWER, NOAA), and are fully reproducible.
